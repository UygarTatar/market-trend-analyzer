# Agentic Digital Market Trend Analyzer

An end-to-end agentic AI system that autonomously collects, analyzes, and reports on trends across:
- 📱 **Mobile Apps** (Google Play + App Store)
- 🎮 **Mobile Games** (all major genre categories)
- 🖥️ **PC Games** (Steam top sellers)

## Setup

1. Clone the repo
2. Copy `.env.example` to `.env` and fill in your API keys
3. Install dependencies: `pip install -r requirements.txt`
4. Run locally: `python app.py`
