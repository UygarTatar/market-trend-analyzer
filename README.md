# 🌌 ONYX: Agentic Market Intelligence System

An autonomous, end-to-end data engineering platform designed to scout, analyze, and report on digital market opportunities across the global gaming ecosystem.

---

### 🚀 Live Features
- **Autonomous Scouting**: Nightly scrapers for **Steam**, **Google Play**, **Apple App Store**, and **Reddit**.
- **Trend Scoring Engine**: A weighted analysis algorithm that detects "Viral Velocity" across platforms.
- **ReAct Reasoning Agent**: Powered by **Gemini 3.1 Flash**, the agent performs real-time market research and strategic opportunity mapping.
- **Self-Healing Pipeline**: Integrated GitHub Actions for automated daily data collection and cloud synchronization.

### ⚙️ Installation & Usage

1. **Clone the Intelligence**:
   ```bash
   git clone https://github.com/UygarTatar/market-trend-analyzer.git
   ```
2. **Configure Secrets**:
   Copy `.env.example` to `.env` and insert your `GOOGLE_API_KEY`.
3. **Initialize Environment**:
   ```bash
   pip install -r requirements.txt
   ```
4. **Deploy Locally**:
   ```bash
   python app.py
   ```
